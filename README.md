# React playground

This project was a great opportunity on my way to learn React.js

## Install
```
git clone git@github.com:szilveszter9/react-playground.git
cd react-playground
npm install
npm run bundle
# on a separate terminal
npm start
```
If any issues, try with node: v4.2.6 and npm: 3.8.3
